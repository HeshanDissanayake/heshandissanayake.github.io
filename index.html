FPGA CPU implementation

A verilog based 8 bit computer to implement on a FPGA. This uses 32 bit instructions and consists of Data memory, instruction memory, ALU, Register file, and control unit modules. the data and instruction memory has the facility of caching that reduces the memory fetching time expence.
